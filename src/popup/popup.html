Режим «Легкий»
  1. 1.Запускайте тільки на сторінці вашого АІ — додаток має бути постійно відкритим.
  2. 2.Отримайте АПІ ключ (як отримати) та додайте його на головному екрані.
  3. 3.Оберіть папку, натиснувши «Обрати / Оновити».
  4. <ваша папка>
    └── Ваша модель АІ/
        ├── full/ → повні тексти
        ├── short/ → стислі записи
        ├── _system/ → службові сесії
        ├── memory.txt → зміст архіву
        ├── tags.txt → файл тегів
        └── books/combined/ → книга розмов
    4. Дочекайтесь завантаження всіх сесій та натисніть «Старт».
    5. Після створення архіву натисніть «Створити книгу».
